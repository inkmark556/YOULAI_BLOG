
# CODE PHANTOM | 博客系统

## 项目简介

本项目是一个基于 Node.js 的个人技术博客系统，前端采用原生 HTML/CSS/JS，后端使用 Express，支持文章管理、分类筛选、分页、编辑与删除、隐藏编辑器等功能。适合技术笔记、项目分享、游戏开发日志等场景。

## 代码目录结构

```
├── index.html              # 首页，文章列表与分类筛选
├── post.html               # 文章详情页，支持 Markdown 渲染
├── editor.html             # 文章编辑/新建页面
├── server.js               # Node.js 后端服务，API 路由
├── package.json            # 项目依赖与脚本
├── posts.json              # 文章元数据列表（由后端维护）
├── .gitignore              # Git 忽略文件
├── posts/                  # 文章内容（Markdown 格式）
│   ├── unity_001.md
│   ├── unity_002.md
│   ├── unity_003.md
│   ├── about.md            # 个人介绍页
│   └── ...                 # 其他文章
├── css/                    # 样式文件
│   ├── index.css           # 首页/列表页样式
│   ├── editor.css          # 编辑器页面样式
│   └── ...
├── js/                     # 前端 JS 文件
│   ├── index.js            # 首页/列表页逻辑（分页、筛选、删除等）
│   ├── editor.js           # 编辑器页面逻辑（新建/编辑/AI填充等）
│   └── ...
```


## 功能说明

- 文章列表展示、分类筛选、分页浏览
- 文章详情页支持 Markdown 渲染
- 文章新建、编辑、删除（带弹窗确认）
- 隐藏编辑器（A-B-A-B-上-下-上-下 组合键触发）
- 文章数据存储于 `posts.json`，内容存储于 `posts/` 目录
- 后端 API 支持上传、删除、AI自动摘要等
- 响应式布局，适配 PC 与移动端


## 快速启动

1. 安装依赖
	```bash
	npm install
	```
2. 启动服务
	```bash
	node server.js
	```
3. 浏览器访问
	```
	http://localhost:3000
	```


## 文章发布与管理

- 普通用户仅浏览，无需登录
- 管理员可通过隐藏编辑器（A-B-A-B-上-下-上-下 组合键）发布新文章
- 支持文章编辑、删除（带弹窗确认，数据实时同步）
- 新文章自动写入 `posts.json` 并保存 Markdown 内容到 `posts/` 目录


## 主要文件说明

- `index.html`：博客首页，文章列表、分类筛选、分页控件
- `post.html`：文章详情页，URL 参数 `id` 加载 Markdown 内容
- `editor.html`：文章新建/编辑页面，支持 AI 自动填充
- `server.js`：Node.js 后端，API 路由、数据同步
- `posts.json`：文章元数据列表，包含 id、标题、日期、标签、摘要等
- `posts/*.md`：每篇文章的 Markdown 内容文件
- `css/`、`js/`：前端样式与脚本


## 特色功能

- 文章分类筛选，标签可自定义扩展
- 分页浏览，支持大量文章高效管理
- 文章编辑/删除，数据实时同步
- 隐藏编辑器带弹性动画，支持 Markdown 编辑与预览
- 作弊码触发隐藏编辑器，提升趣味性与安全性
- 响应式设计，适配多终端
- 侧边栏展示作者信息、联系方式（含 QQ）、标签云
